## JDK 12 ~ 14
* G1 收集器优化
* SocketAPI 重构
* 动态 CDS 存档
* 增强 switch

* 移除 CMS 垃圾收集器

* """ 标识文本块 """
* instanceof 模式匹配
  * 之前 a instanceof Person之后需要(Person)a，现在不需要了
* Hidden Classes（隐藏类）
  * 运行时生成类文件，只能通过反射访问
* 关键字record
  * final类
  * 低配的lombok.@Data
* ZGC 并发线程堆栈处理
